# FSV Belegungsplan

Eine Schritt-für-Schritt-Anleitung zur Einrichtung findest du in der Datei
**"Anleitung Belegungsplan einrichten.docx"**.

## Kurzfassung

1. Supabase-Projekt anlegen, `supabase-setup.sql` im SQL-Editor ausführen
2. Projekt-URL und anon-key aus Supabase in `src/supabaseClient.js` eintragen
3. `npm install`
4. Lokal testen: `npm run dev`
5. Auf GitHub hochladen, mit Vercel verbinden, deployen
